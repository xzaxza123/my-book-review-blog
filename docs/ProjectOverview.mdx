## 项目是什么

My Book Review Blog 是一个专注长文阅读体验的前端项目。目标是在浏览器里模拟纸质书的翻页感受，让技术文章和读书笔记以书本的形式被阅读。

项目采用 React、TypeScript、Vite 和 MDX 作为基础技术栈，同时在分页算法和书本翻页动画上做了相对完整的一套实现，可以作为展示前端工程能力的作品。

## 核心体验

- 书本翻页式阅读
- 面向 DOM 的分页算法
- MDX 驱动的文档系统
- 针对桌面端和移动端的自适应布局

这些能力共同服务于一个目标：让阅读长文变得顺滑、连续、不被滚动条打断。

## 适合做什么

- 个人技术博客或读书笔记
- 展示复杂交互和前端工程细节的作品集项目
- 探索基于 MDX 的文档与阅读体验

如果你正在寻找一个既有视觉效果又有工程深度的前端项目，这个书本阅读博客可以作为不错的基础。

## 体验路径

阅读体验通常可以拆成三个阶段。

### 进入项目

首页会展示项目封面、简单介绍以及入口按钮。根据设备尺寸不同，首页会在布局和字号上做一定调整，使内容在桌面和移动设备上都清晰可读。

### 打开书本

点击进入阅读页后，你会看到一本居中的翻页书。每一页都是分页算法计算出来的物理页面，支持翻页动画和页码显示。

你可以通过点击页角、在移动端滑动或者使用控制按钮来翻页。

### 深入阅读

正文内容来自 docs 目录下的 MDX 文档。文章会被渲染到隐藏容器，再由分页算法拆分成实际的页面。你看到的每一页，都已经过高度测量、图片缩放和布局调整。

当内容包含图片、代码块、表格和引用等复杂元素时，阅读体验依旧保持稳定，不会出现大块空白或内容被截断的情况。

## 项目亮点

- 兼顾观感和可维护性的 DOM 分页算法
- 支持图片、代码、表格、列表、引用和自定义布局的多类型分页
- 明确的模块分层，核心逻辑集中在少数几处
- 基于 MDX 的内容系统，便于扩展新的文档和文章

这些亮点在后续的架构文档和分页算法文档中会有更细致的说明。
